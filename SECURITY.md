# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | Yes |

## Reporting a Vulnerability

Email **security@atomicstrata.ai** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Affected package version, if known

Do not open a public GitHub issue for security vulnerabilities. We respond within 48 hours.
